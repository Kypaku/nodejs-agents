# Node GPT TDD Agents


## Environment

Node 16.19

VueCli 4

## Project setup

```
git clone https://github.com/Kypaku/vue-gpt-agents
```

```
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Compiles and minifies for production
```
npm run electron:build
```
